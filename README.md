# JS-game

# 简介
用js结合Jquery和underscore两个库完成的简单小游戏：炸弹人（QQ堂精简版），利用了面向对象的思想和中介者模式，全部动态效果使用全局定时器实现。大概是某次没有网的自娱自乐？？

# 玩法说明
* 管理员注册和管理
1、空格放炸弹、炸掉附近元素；
2、角色能推动可移动元素；
3、有三种道具可以拾取，可增加角色相应的属性；
4、每个地图有一定数目的怪物数量，怪物可拾取道具，拾取了道具的怪物被炸死后道具会被炸出来；
5、 被炸死为失败，完成任务即可过关；
6、四种角色、四种炸弹和四种地图可选
